마루 – AI 런처, 홈 화면의 새로운 중심
홈 화면이 복잡하고 앱 찾기가 번거로우셨나요?
마루 런처앱은 설치된 앱을 자동으로 분류하고, 첨단 온디바이스 인공지능를 홈 화면에서 바로 사용할 수 있도록 도와줍니다. 또한 인스타그램 최신 게시글, 틱톡 인기 영상, 최신 뉴스까지 한눈에 확인할 수 있어 스마트 홈 화면 경험을 제공합니다.
주요 기능
앱 자동 정리: 설치된 앱을 용도별로 분류해 효율적인 홈 화면 구성
AI 런처: Gemma-4 기반의 온디바이스 인공지능
소셜 피드: 인스타그램 최신 게시글과 틱톡 인기 영상 확인
뉴스 위젯: 최신 뉴스 소식을 홈 화면에서 빠르게 확인
맞춤형 홈 화면: 사용자 라이프스타일에 맞춘 최적화된 런처 경험
