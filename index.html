  • 설치와 동시에 실시간 감지

서비스 이용약관

제 1 조 (목적)

본 약관은 MARU Ecosystem(이하 "회사")이 제공하는 온디바이스 AI 런처 "MARU"(이하 "앱" 또는 "서비스")의 이용과 관련하여 회사와 회원 간의 권리, 의무 및 책임사항을 규정함을 목적으로 합니다.

제 2 조 (온디바이스 서비스의 특성 및 정의)

1. MARU는 사용자 기기 내부에서 직접 작동하는 경량 인공지능(Gemma-4 E2B LiteRT-LM)을 기본 엔진으로 장착하여 오프라인 환경에서도 지능형 비서 역할을 수행하는 '온디바이스 비서 생태계'입니다.
2. 본 서비스에서 이루어지는 인공지능 자연어 추론, 자동 카테고리 분류, 확장자 추론 검색, 하드웨어 제어 등은 기기 자체의 연산 자원만을 이용하여 로컬에서 안전하게 구동됩니다.

제 3 조 (회사의 의무와 안전장치)

1. 회사는 인공지능 환각 현상(Hallucination)으로 인한 사용자의 데이터 소실을 예방하기 위해 '사용자 승인 기반 안전 제어(Human-in-the-Loop)' 인터페이스를 기본 탑재합니다.
2. 파일 삭제, 이름 변경 등 시스템의 매개변수를 바꾸는 중요 행위 시 최종 동작은 오직 사용자의 실질적인 손가락 터치 승인이 있을 때에만 하드웨어 레벨에서 정상 처리됩니다.

제 4 조 (이용자의 권리와 의무)

1. 회원은 MARU 앱을 안드로이드 9.0 이상의 적합한 시스템 요구사항이 만족되는 기기에서 안전하게 사용할 권리를 가집니다.
2. 회원은 AI 스티커 공방 등에서 제작한 본인의 창작 이미지 데이터를 홈 화면에 마음대로 배치하고 활용할 수 있습니다.
3. 회원은 애플리케이션과 함께 제공되는 일체의 데이터(홍보 영상, UI 에셋, 이미지, 텍스트, 소스코드 및 기타 유관 리소스 등)를 회사의 공식적인 사전 서면 동의 없이 독단적으로 배포, 무단 공유, 편집, 수정, 복제하거나 상업적인 목적으로 재사용 및 가공해서는 안 됩니다.

개인정보처리방침

1. 수집하는 개인정보 항목 및 보관 방법

MARU는 강력한 온디바이스(On-Device) 구동 원칙에 따라 설계되었습니다. 사용자의 어떠한 사생활 정보나 텍스트 입력값, 로컬 검색 기록 및 음성 분석 데이터도 외부 서버로 수집·전송하지 않습니다.
- 앱 자동 분류 정보: 로컬 기기 내에서만 스캔되며, 데이터베이스(Room DB)에 암호화 보관됩니다.
- 파일 에이전트 탐색 경로: 기기 메모리 상에서 실시간으로만 매칭 후 바로 휘발됩니다.
- AI 스티커 공방 및 미디어 리소스: 오직 사용자의 로컬 단말 내 Room DB 및 갤러리 영역에 안전하게 저장됩니다.

2. 개인정보의 제3자 제공

회사는 사용자의 정보를 수집하지 않으므로 제3자에게 임의로 가공·판매·임대하거나 양도하지 않습니다. 모든 추론 연산과 처리는 사용자가 소유한 기기의 하드웨어 내부에서 완전 종결됩니다.

3. 안전성 확보를 위한 기술적 조치

MARU는 구글 플레이스토어 보안 가이드를 완벽하게 준수하며, 시스템의 깊은 접근 권한을 요구하는 경우에도 오직 오프라인 NLU 추론에만 사용됩니다. 외부 전송 파이프라인 자체가 존재하지 않으므로, 해킹으로 인한 서버 단에서의 사용자 데이터 대량 유출 우려가 구조적으로 차단되어 있습니다.

보안 및 안전 가이드

🔐 MARU의 3중 레이어 보안 프로토콜

우리는 인공지능이 기기를 제어함에 있어 발생할 수 있는 잠재적 위협과 오작동을 완벽히 방지하기 위해 정교한 보안 기술을 탑재했습니다.

1. Human-in-the-Loop UX (최종 결정권 보장)

AI가 아무리 합리적인 추론을 통해 "이름이 겹치는 중복 파일 삭제"를 결정하더라도, 안드로이드 Kotlin Native 파일 제어 단(delete(), renameTo())은 오직 사용자가 직접 모니터에 노출된 [승인] 버튼을 누른 찰나에만 활성화됩니다. 독단적 파괴 명령이 원천 봉쇄됩니다.

2. Local Room DB Sandbox

사용자 설정 및 홈 화면 스티커 배치, 드래그 위치 등의 메타데이터는 샌드박스화된 로컬 전용 룸 데이터베이스에 저장됩니다. 이는 타 루트 권한 앱이나 외부 네트워크가 함부로 침범할 수 없는 안전지대입니다.

3. 원격 라이프사이클 관리

GitHub Gist를 활용한 캐시 무력화 통신 채널을 결합하여, 심각한 제로데이 취약점 감지 시 즉각적으로 원격 라이프사이클을 가동, 안전한 패치가 이루어질 때까지 취약 버전을 완벽하게 차단하고 통제합니다.
